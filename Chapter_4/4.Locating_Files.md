# Locating Files

Có 1 số lệnh được sử dụng để tìm các tệp là: 

`whereis`, `find`
 
- `localte`

Để tải về locate: 

```
yum install -y mlocate
```

Khi chuyển 1 vị trí mới, ta phải thực hiện updatedb để cập nhật dữ liệu các file và thư mục. 

```
updatedb
```
